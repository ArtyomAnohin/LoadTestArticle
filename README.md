# Load Test Article #

Example project to the article Load Test for the site [agilites.com/](https://agilites.com/)

### Contains a multi-module project ###

##### -gatling
Server side testing
##### -timingAPI
Client side testing


### To run use gradle tasks: ###
##### Server Side: #####
`gatlingRun-<Test_Name>`

##### Client Side: #####
`chrome` or `firefox`
